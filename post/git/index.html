github 多账户使用ssh

ssh-keygen -t rsa -C "wuguojun1024@gmail.com" -f ~/.ssh/id_rsa_wugjun

ssh-keygen -t rsa -C "864785862@qq.com" -f ~/.ssh/id_rsa_drsin

启动ssh代理

eval ssh-agent -s

添加config配置文件分别映射不同的GitHub账户

进入~/.ssh目录，新建config文件，并添加相应的内容：

# drsin GitHub
Host github.com
HostName github.com
PreferredAuthentications publickey
# 指定特定的ssh私钥文件
IdentityFile ~/.ssh/id_rsa_drsin

# wugjun GitHub  别名
Host wugjun 
HostName github.com
PreferredAuthentications publickey
# 指定特定的ssh私钥文件
IdentityFile ~/.ssh/id_rsa_wugjun
